Setup & Installation

Follow these steps to set up and install the project:

  1. Download: Get the latest release from the official download page.
  2. Extract: Unzip the downloaded file to your preferred directory.
  3. Install Dependencies: Open a terminal in the project folder and run:
    npm install
  4. Configure: Edit the config.json file to match your environment.
  5. Run the Application: Start the program with:
    npm start

For detailed instructions, see the documentation or contact support if you encounter issues.

About

Mars Preserve is dedicated to sustainable farming using advanced climate containers. Our mission is to empower individuals and communities to grow food efficiently and responsibly.

Project Overview

The platform provides real-time monitoring, automated climate control, and analytics for container-based agriculture. It is designed for scalability, security, and ease of use.

Main Modules

  • Monitoring: Track temperature, humidity, CO2 levels, and other vital stats in real time.
  • Control: Remotely adjust climate settings, lighting, and irrigation for optimal crop growth.
  • Analytics: Visualize historical data, generate reports, and receive insights for better decision-making.
  • User Management: Manage user roles, permissions, and access to different containers.
  • Notifications: Get alerts for critical events, maintenance reminders, and system updates.
  • Support & Documentation: Access help resources, guides, and direct support from the team.

The project is open for collaboration and welcomes feedback to improve features and usability.
